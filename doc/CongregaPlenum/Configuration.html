    class CongregaPlenum::Configuration

    Shared configuration used by the client and services to tweak base URL, timeouts and retry policies. Defaults focus on the public APIs hosted by the Câmara so a regular application can simply call {CongregaPlenum.configure} and override what differs (timeouts, logger, etc.).

    Attributes

    base_url [RW]

    @return [String] Base endpoint for all requests.

    logger [RW]

    @return [Logger] Logger used across adapters/services.

    rate_limit_delay [RW]

    @return [Float] Delay between paginated calls to respect rate limits.

    retries [RW]

    @return [Integer] How many times requests are retried on transient errors.

    retry_delay [RW]

    @return [Float] Initial backoff delay in seconds.

    timeout [RW]

    @return [Integer] Timeout (seconds) applied to open/read operations.

def initialize
  @base_url = 'https://dadosabertos.camara.leg.br/api/v2'
  @timeout = 30
  @retries = 3
  @retry_delay = 1.0
  @rate_limit_delay = 0.1
  @logger = Logger.new($stdout)
end

    Builds a configuration instance with safe defaults tuned for the Câmara APIs. Those endpoints are known to be HTTPS-only, relatively slow, and rate-limited, so the defaults try to smooth that out.
